SeelViz: Data Visualization of Clarity Brain Images

SeelViz (short for Clarity Visualization) is a collection of tools used to analyze and visualize CLARITY-treated brains. This website is maintained to provide deliverables (see here) of our weekly progress as well as extensive documentation regarding our projects. Upload CSV files here to perform processing on our web server (in alpha, please upload 200 kb or less).

Graph Explanations

Shown above is an example output of a density false-colored for Cocaine 174 generated via ClarityViz. The colorations correspond to the number of edges for each node (an edge-number-to-color mapping can be seen here. Each unique color represents a 5% increase in total number of nodes, with the most-connected node being the white node. The original source code can be seen here.
